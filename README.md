# Mapa – Casa Julián Alvarez

Interactive illustrated map of Palermo, Buenos Aires.

## Image files required

Place the following PNG exports from Canva inside the `images/` folder:

| Filename | Content |
|---|---|
| `map-bg.png` | Full background map |
| `malba.png` | Building illustration – Malba |
| `teatro-colon.png` | Building illustration – Teatro Colón |
| `ninina.png` | Building illustration – Ninina |
| `chui.png` | Building illustration – Chuí |
| `el-gordo-chanta.png` | Building illustration – El Gordo Chanta |
| `el-preferido.png` | Building illustration – El Preferido |
| `la-ideal.png` | Building illustration – La Ideal |

Export each at **4x** resolution from Canva (File → Download → PNG → Scale 4x).

## Deploy

Deploy this repo as a static site on [Vercel](https://vercel.com). The `/mapa` route will serve the page automatically via `vercel.json`.
